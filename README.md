# olive-portal

おりーぶ庵株式会社の人事・労務DXシステムです。

## Ver.0.1でできること

- Apple風・白ベースのスマホ対応画面
- 従業員の新規登録
- 従業員情報の編集
- 従業員検索と雇用形態の絞り込み
- 退職処理と退職者一覧
- 健康診断日・扶養控除等申告書の簡易確認
- ブラウザ内への仮保存

## 重要

Ver.0.1は動作確認用です。入力データはブラウザのLocalStorageに保存されます。

個人番号、銀行口座、健康情報などの機微情報は入力しないでください。
本番運用前にGoogle Apps Script、Googleスプレッドシート、権限管理、監査ログ、バックアップを実装します。

## Netlify

公開ディレクトリはリポジトリのルートです。
ビルドコマンドは不要です。
